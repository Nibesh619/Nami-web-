Here, we were given an assignment where we have to make our own website using only HTML and CSS. In this website, 6 pages were linked with each other i. e. home, about, C.V, social, contact and model respectively. In home page, personal photo was given and navigational bar were shown. Similarly, in about page personal description were given. In this page, visitor of this website can see my habit, aim and so on. In C.V page, my academic background were mentioned as well as my bio-information are displayed. In social page, my social sites are displayed were people can see my day to day activity in life. In contact page, people or user can contact me by giving me some feedback. And last but not the least, model page is for report writing, This website represents the profile of developer and gives personal information. This website has good animation and good color combination. It is very simple website and very basic html and css code were written.
